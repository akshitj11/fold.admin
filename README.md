# Fold Admin Dashboard

React + TypeScript + Vite admin dashboard for Fold Web3 infrastructure monitoring.

## Setup

```bash
npm install
```

Create a `.env` file:

```env
VITE_API_URL=http://localhost:3000
VITE_ADMIN_WALLET_ADDRESS=
VITE_MEMORY_VAULT_ADDRESS=
VITE_PAYMASTER_ADDRESS=
VITE_PREMIUM_SBT_ADDRESS=
VITE_STORAGE_NFT_ADDRESS=
VITE_NETWORK=testnet
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
